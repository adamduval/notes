### components of css

```css
h1{
    color: blue;
    text-align: right;
}
```

- entire thing is a rule
- h1, selector
- {}, declaration block
- color: blue, individual declaration
- color, property
- blue, value

A CSS rule is a block of declarations which apply declaration properties and values to a selector.
